## Class and Objects
**Class**
* Class is a collection of objects, classes are blueprints for creating objects. Class defines the set of attributes and methods that the created objects(instances) can have
**Object**
* An Object is an instance of a Class. It represents a specific implementation of the class and holds its own data.

## 6-Week Plan

### Week 1 (Mar 13-19): LangGraph + Application Launch
- **Build:** Complete Bubble Phase 3 — LangGraph agentic orchestration, tool calling, MemorySaver
- **Apply:** Update resume for AI App Engineer. Start applying 3-5/day
- **DSA:** Arrays & Hashing — Contains Duplicate, Two Sum, Group Anagrams, Top K Frequent
- **Target:** LangGraph phase done + 15-20 applications sent

### Week 2 (Mar 20-26): Streaming + Portfolio Blitz
- **Build:** Complete Bubble Phase 4 — real streaming, Streamlit UI polish
- **Apply:** Continue 3-5/day. Start LinkedIn outreach
- **DSA:** Two Pointers + Sliding Window
- **Target:** Bubble deployed + demo video + 30-40 total applications

### Week 3 (Mar 27-Apr 2): Docker + Deploy
- **Build:** Dockerize Bubble. Deploy to Railway/Render
- **Apply:** Follow up on applications. Should start getting responses
- **DSA:** Stack + Binary Search
- **Target:** Dockerized + 50+ applications + first responses

### Week 4 (Apr 3-9): Second Project + Interview Prep
- **Build:** Small second project (2-3 days) — MCP server OR AI agent
- **Apply:** Prioritize companies that responded
- **DSA:** Trees + Graphs basics
- **Target:** Second project done + interview pipeline active

### Week 5 (Apr 10-16): Interview Sprint
- **Focus:** Interviews take priority over everything
- **Target:** Multiple interviews in progress

### Week 6 (Apr 17-23): Decision Week
- **Evaluate offers vs. staying at PhonePe**
- **Target:** Offer(s) in hand or strong pipeline
